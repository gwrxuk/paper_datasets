Distribution Analysis

Distribution Analysis Methodology

The distribution analysis was performed using the following steps:

  1. Sentiment Score Calculation:
    • Each response was analyzed using TextBlob's sentiment analyzer
    • Scores range from -1.0 (most negative) to 1.0 (most positive)
    • The formula used: sentiment_score = TextBlob(text).sentiment.polarity
  2. Distribution Visualization:
    • Histograms were created using seaborn's histplot function
    • Kernel Density Estimation (KDE) curves were added to show the probability density
    • The KDE formula: KDE(x) = (1/nh)∑K((x-x_i)/h)
    • Where K is the kernel function, h is the bandwidth, and n is the number of data points
  3. Distribution Statistics:
    • Mean (μ): μ = (1/n)∑x_i
    • Standard Deviation (σ): σ = √[(1/n)∑(x_i - μ)²]
    • Skewness: γ = (1/n)∑[(x_i - μ)/σ]³
    • Kurtosis: κ = (1/n)∑[(x_i - μ)/σ]⁴ - 3
  4. Interpretation Guidelines:
    • Normal Distribution: Skewness ≈ 0, Kurtosis ≈ 0
    • Positive Skew: Right-tailed distribution
    • Negative Skew: Left-tailed distribution
    • High Kurtosis: Heavy tails, more outliers
    • Low Kurtosis: Light tails, fewer outliers

Effect Size Analysis

Small Effect Size (d = 0.2)

A small effect size indicates subtle differences between responses. This level of effect suggests:

  • Minor variations in response patterns
  • Small but noticeable differences in politeness levels
  • Subtle changes in emotional expression

Medium Effect Size (d = 0.5)

A medium effect size represents more substantial differences between responses. This level indicates:

  • Moderate changes in response characteristics
  • Notable differences in politeness strategies
  • Significant variations in emotional content
